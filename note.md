#### 字串

• 用 ' 、 '' 包起來的就是字串

• 單引號 和 雙引號之前沒有差別
• 特殊字元:

\n 換行
\t 4~8個空格(排版用)
• a = "1234567890"

• 其他操作:a[從(包含):到(不包含)(:間隔)]

• a[:3] : "123"

• a[-3:] : "890"

• a[2:-4] : "3456"

• a[::-1] : "0987654321"

• 從 0 開始算
• 常用相關內建函式:

str() 把一堆鬼東⻄都轉成字串 : str(123) → ‘123'
chr() 把 Ascii 值轉字元 : chr(0x41) → 'A'
ord() 取得字元 Ascii 值 : ord('A') → 65
len() 取得字串長度 : len('ABC') → 3
